# 安全管理端 - 生产环境安全检查清单

在将系统部署到生产环境之前，请务必完成以下所有检查项。

## 🔐 认证与授权

- [ ] JWT Secret 已更换为强随机字符串（至少 64 位）
- [ ] JWT 过期时间已设置为合理值（建议 2-8 小时）
- [ ] 默认管理员密码已修改
- [ ] 已删除或禁用测试账号
- [ ] 已配置合理的角色和权限
- [ ] 超级管理员账号已妥善保护

## 🗄️ 数据库安全

- [ ] 数据库密码已更换为强密码
- [ ] 已启用 SSL/TLS 连接（sslmode=require）
- [ ] 数据库访问已限制特定 IP
- [ ] 已创建专用数据库用户（非 postgres）
- [ ] 已配置定期备份策略
- [ ] 数据库端口未暴露到公网

## 🔑 Redis 安全

- [ ] Redis 已设置访问密码
- [ ] Redis 访问已限制特定 IP
- [ ] 已禁用危险命令（FLUSHALL, CONFIG, KEYS 等）
- [ ] Redis 端口未暴露到公网
- [ ] 已配置持久化策略

## 🌐 网络安全

- [ ] 已配置 HTTPS/TLS
- [ ] 已使用反向代理（Nginx 等）
- [ ] 已配置防火墙规则
- [ ] 已启用 IP 白名单（如适用）
- [ ] CORS 配置已限制为具体域名
- [ ] 已禁用不必要的 HTTP 方法

## 📝 日志与审计

- [ ] 日志级别已设置为 warn 或 error
- [ ] 审计日志已启用
- [ ] 日志文件权限已正确设置
- [ ] 已配置日志轮转和保留策略
- [ ] 审计日志已定期审查
- [ ] 敏感信息不会记录到日志中

## 🔒 应用安全

- [ ] 运行模式已设置为 release
- [ ] 密码策略已启用并配置合理
- [ ] 登录失败限制已启用
- [ ] 限流保护已启用
- [ ] 请求体大小限制已配置
- [ ] 超时设置已配置合理

## 🛡️ 系统安全

- [ ] 操作系统已更新到最新版本
- [ ] 已安装并配置防火墙
- [ ] 已禁用不必要的系统服务
- [ ] 已配置安全更新自动安装
- [ ] 已部署入侵检测系统（可选）
- [ ] 已配置安全监控和告警

## 📦 依赖安全

- [ ] 所有依赖已更新到最新稳定版本
- [ ] 已检查依赖的安全漏洞
- [ ] 已移除不必要的依赖
- [ ] go.sum 文件已提交到版本控制

## 🚀 部署安全

- [ ] 配置文件未包含在版本控制中
- [ ] 敏感信息使用环境变量或密钥管理服务
- [ ] Docker 镜像已扫描安全漏洞
- [ ] Kubernetes Secret 已加密
- [ ] 已配置资源限制（CPU/内存）
- [ ] 已配置健康检查和自动重启

## 🔄 运维安全

- [ ] 已制定应急预案
- [ ] 已配置监控和告警
- [ ] 已制定备份恢复策略
- [ ] 已进行灾难恢复演练
- [ ] 已建立安全事件响应流程
- [ ] 已定期进行安全审计

## 📋 合规性

- [ ] 符合数据保护法规要求
- [ ] 已制定隐私政策
- [ ] 用户数据已加密存储
- [ ] 已建立数据访问审批流程
- [ ] 已定期进行合规性检查

## ✅ 测试验证

- [ ] 已进行渗透测试
- [ ] 已进行压力测试
- [ ] 已进行安全扫描
- [ ] 所有测试问题已修复
- [ ] 已进行代码审查

## 📞 紧急联系

- [ ] 已建立安全事件联系人列表
- [ ] 已配置紧急告警通知
- [ ] 已制定应急响应预案
- [ ] 已定期进行应急演练

---

## ⚠️ 重要提醒

1. **永远不要**将包含密码、密钥的配置文件提交到版本控制
2. **永远不要**使用默认密码或弱密码
3. **永远不要**在生产环境使用 debug 模式
4. **永远不要**将数据库或 Redis 端口暴露到公网
5. **一定要**定期更新系统和依赖
6. **一定要**定期审查审计日志
7. **一定要**定期备份数据
8. **一定要**进行安全测试
